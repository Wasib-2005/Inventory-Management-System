This is a temporary README for Git. Replace with full documentation later.

## About

- Minimal inventory management system with separate `Backend` and `Frontend`.

## Repo structure

- `Backend/` — Node.js API server (Express)
- `Frontend/` — Client app (Vite + React)
- `certs/`, `keys/` — TLS and key material

<h2 style="color:red">Quick start</h2>

<p style="color:red">Prerequisites: Node.js 16+ and <code>npm</code></p>

<p style="color:red"><strong>Backend:</strong></p>
<pre style="color:red"><code>cd Backend
npm install
# start server (development)
node server.js
</code></pre>

<p style="color:red"><strong>Frontend:</strong></p>
<pre style="color:red"><code>cd Frontend
npm install
</code></pre>

<p style="color:red"><strong>Development (dev):</strong></p>
<pre style="color:red"><code># Dev: run commands (do not run now)
# cd Frontend
# npm run dev
</code></pre>

## Environment

- Backend uses an `.env` file located in the `Backend` folder — review `Backend/.env`
- Secrets and private keys are stored in `Backend/keys` (do NOT commit)
